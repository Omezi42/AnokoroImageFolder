🔑 GitHub Personal Access Token (PAT) の取得方法

  1. こちらのリンク をクリックしてGitHub設定を開きます。
  2. Noteには「AnokoroImageProcessor」等と入力され、repoスコープにチェックが入っていることを確認します。
  3. 一番下の [Generate token] をクリックします。
  4. 表示された ghp_... から始まる文字列をコピーし、下の入力欄に貼り付けて「保存」を押してください。
